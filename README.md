# MK TypeSprint

**Typing Speed Test & Practice Platform**

Measure and improve your typing speed and accuracy. Practice with words, quotes, code, and numbers.

🔗 **Live**: [https://11-web-keyboard-practice.vercel.app](https://11-web-keyboard-practice.vercel.app)

## Tech Stack

Vanilla HTML/CSS/JS, Vite 7

## Getting Started

```bash
npm install
npm run dev
```
