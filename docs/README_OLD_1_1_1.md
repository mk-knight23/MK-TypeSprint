## 🚀 Quick Start

### Prerequisites

- Node.js 22+ and npm

### Installation

```bash
# Clone the repository
git clone https://github.com/mk-knight23/11-web-keyboard-practice.git
cd 11-web-keyboard-practice

# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:3000` (or check console output).

## 🧪 Testing

```bash
# Run tests
npm test

# Run tests with coverage
npm run test:coverage
```

## 🏗️ Build

```bash
npm run build
```
