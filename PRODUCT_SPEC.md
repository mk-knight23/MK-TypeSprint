# Product Specification - MK TypeSprint

- **Core Problem:** Users need a simple, fast, free, registration-free utility to solve: "Typing practice and performance platform".
- **Target Audience:** Everyday internet users, developers, and professionals.
- **Key Workflows:**
  1. Instant utility access on page load (simple first).
  2. Advanced configuration options via drawers/tooltips (progressive disclosure).
  3. Safe localStorage or IndexedDB persistence.
- **Features Included:** Timed tests, word/quote/coding modes, real-time WPM/accuracy tracking, heatmap, practice history.
